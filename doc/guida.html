1. Propositi

Questo documento contiene informazioni sull'installazione e sull'utilizzazione del simulatore per la  microarchitettura mic1 e delle relative  funzionalita'.  Per poter utilizzare al meglio il simulatore e' opportuno riferirsi al testo Andrew S. Tanenbaum, Structured Computer Organization, 4th Edition, (Prentice-Hall 1999), di cui ne esiste anche una versione in italiano, ed in particolare al "Chapter 4: The Microarchitecture Level."

Il software e' stato realizzato completamente in Java (JDK V1.0 e successivi). I file sorgenti per i programmi sono inclusi nel pacchetto contenente il presente documento e forniscono agli utenti un mezzo per implementare altre microarchitetture ed altri assemblatori e microassemblatori.

Si fa notare che il software e' stato concepito per essere utilizzato con sistemi operativi molto diversi tra loro (Solaris, Linux, Unix e Windows per citare i piu' conosciuti): puo' essere quindi definito portabile (grazie soprattutto alle caratteristiche di Java).

2. File

Sono presenti diversi formati di file (caratterizzati dall' estensione) dovuti sia ai diversi sitemi operativi per cui il simulatore e' stato concepito, sia ai nuovi file usati dal simulatore stesso. 

Puo' quindi essere utile una panoramica  sui file utilizzati: 

.bat

un file batch per Win95/98/NT/00; e' uno script usato per eseguire programmi sotto il sistema operativo Microsoft Windows.

.conf

un file di configurazione; questo e' un file di testo contenente informazioni di configurazione per un programma.

.ijvm

un file di linguaggio Integer Java Virtual Machine; questo è un file binario che contiene il codice oggetto IJVM.

.jas

un file di linguaggio assembly di Java; questo è un file di testo che contiene il sorgente IJVM.

.mal

un file di linguaggio di micro assembly; questo è un file di testo che contiene il codice sorgente per un microprogramma.

.mic1

un microprogramma per l'architettura mic1; questo è un file binario che contiene il codice oggetto mic1.

File in Java per il programma

·mic1asm, un assembler per Mic1 Micro-Assembly Language (MAL)

·gmic1asm, un'iterfaccia grafica per l'assembler mic1asm.

·ijvmasm, un assembler per Integer Java Virtual Machine (IJVM)

·gijvmasm, un'iterfaccia grafica per l'assemblerijvmasm.

·mic1sim, un simulatore per l'architettura mic1.

·dump, un programma che stampa il contenuto di un file binario in ottale, decimale ed esadecimale.

·mic1dasm, un disassembler che converte un microprogramma (MIC1) compilato nel linguaggio Micro-Assembly (MAL).

Support files

·ijvm.conf, un file di configurazione per l'assembler ijvmasm. Contiene una descrizione delle istruzioni che include l'opcode, lo mnemonico ed il tipo di operazione di ogni istruzione.

Programmi di esempio

·ijvmtest.jas, a program that tests all the features of the mic1 architecture. 

·echo.jas, a program which accepts keyboard input and prints any key pressed to the standard out text area. 

·add.jas, a program which accepts two hexadecimal numbers from keyboard input and displays the sum. 

·ijvmtest.jas, un programma che testa tutte le caratteristiche dell'architettura mic1.

·echo.jas, un programma che accetta input da tastiera e stampa sulla text area di out ogni lettera premuta.

·add.jas, un programma che accetta due numeri esadecimali immessi da tastiera e ne stampa la somma.
