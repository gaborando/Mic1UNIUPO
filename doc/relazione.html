Modifiche effettuate al codice

Elenco classi già esistenti e modificate:

mic1.ControlStore.java  : E' stato aggiunto un metodo per il settaggio di una singola microistruzione in caso venga modificata a mano dall'utente
mic1.IJVMAssembler.java  : Oltre al file oggetto genera un file .mne che serve per la visualizzazione simbolica della main memory
mic1.IJVMMethod.java : e' stato modificato il metodo generate e aggiunto un metodo generateLabels per scrivere i dati di interesse del metodo sul file .mne
mic1.Mic1Constants.java : Sono stati aggiunti i magic number per i file .mne e la dimensione della porzione di memoria visualizzata di main memory
mic1.MainMemory.java : Si e' aggiunto il settaggio di variabili statiche di mic1.mic1sim che permettono di aggiornare la porzione di memoria visualizzata se questa viene modificata durante l'esecuzione del programma
mic1.Mic1Instruction.java : e' stato aggiunto un metodo per la conversione di una microistruzione in un numero e per settare una microistruzione dato un numero
mic1.MPC.java: E' stato aggiunto il metodo getValue() che ritorna il valore corrente del mpc
mic1.PC.java : E' stato aggiunto il metodo getValue() che ritorna il valore corrente del pc
mic1.mic1sim.java : Nel metodo init() viene aggiunto il menu windows per aprire le finestre(control store, main memory, debug )
Nei vari metodi vengono aggiunte istruzioni per gestire i dati visualizzati dai vari frames se aperti in quel momento.
E' stato aggiunto il metodo macroStep per poter effettuare una macroistruzione alla volta ( controllare se la prima istruzione del microcodice viene sempre  memorizzata all'indirizzo 0x2 nella control store)
Nel metodo loadProgram viene caricato anche il file con estensione .mne

Elenco classi create:

mic1.ControlStoreFrame.java : Frame dove viene visualizzata la control store
mic1.ControlStoreModel.java : Modello dei dati per il frame della control store
mic1.MainMemoryFrame.java : Frame dove viene visualizzata la main memory
mic1.MainMemoryModel.java : Modello dei dati per il frame della main memory
mic1.MNELoader.java : classe che si occupa di caricare il file con i mnemonici per la visualizzazione nel frame della main memory
mic1.OutRangeNumberException.java :
mic1.Simbol.java : classe che essenzialmente implementa una struttura del tipo stringa e indirizzo, utilizzata per la visualizzazione delle label simboliche nel frame della mein memory
